## Problem Statement
### College Event Platform Solution

A first-year engineering student, is determined to address team formation and communication gaps related to college events. His comprehensive platform will feature:

- **Centralized Calendar:** A centralized calendar for scheduling, event creation, categorization, and efficient planning.
- **Streamlined Registration:** Simplified registration processes to encourage student participation.
- **Social Interaction:** Features for commenting, discussion, and feedback on events.
- **Recognition System:** A dynamic leaderboard and achievement system to acknowledge active participants.
- **Automated Event Suggestions:** Intelligent event recommendations based on user interests.
  
The aim is to promote student engagement, facilitate collaboration, and enhance the overall college experience.

## Solution 
* Android App using Jetpack Compose (Kotlin)

Version 1.0 (basic login/signup)
